#  작성자: 방준영

###  시스템 스택 호출과 재귀함수를 이용한 팩토리얼 계산 콘솔 인터렉티브 프로그램 

************************************************************************************
1.  반복문 기반과 재귀 기반의 팩토리얼 계산 함수 구현
2.  유효성 검사 포함 (0 이상 정수 확인)
3.  문자열 입력 → 정수 변환 → 유효성 검사 → 팩토리얼 계산까지 포함된 콘솔 프로그램 형태
4.   q 또는 quit 입력 시 종료
